# 🎬 The Super App

A multi-feature React application that combines user registration, category selection, weather updates, latest news, notes, countdown timer, and entertainment recommendations into a personalized dashboard.

## ✨ Features

### Authentication & Registration
- User Registration
- Name, Username, Email & Mobile Validation
- Inline Error Messages

### Category Selection
- Select Entertainment Categories
- Minimum 3 Categories Required
- Category Selection Validation

### Dashboard
- User Profile
- Weather Widget
- Latest News Widget
- Countdown Timer
- Notes Widget (Local Storage)

### Entertainment Discovery
- Fetch Movies based on Selected Categories
- Movie Cards
- Hover Animations
- Movie Details Modal

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/super-app.git
```

Move into project

```bash
cd super-app
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
VITE_WEATHER_API_KEY=your_weather_api_key
VITE_NEWS_API_KEY=your_news_api_key
VITE_OMDB_API_KEY=your_omdb_api_key
```
